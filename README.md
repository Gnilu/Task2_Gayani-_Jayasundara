# Exploratory Data Analysis (EDA) using Python

This project demonstrates the process of **Exploratory Data Analysis (EDA)** on a retail sales dataset using Python. The analysis focuses on understanding the dataset, identifying trends, detecting outliers, and visualizing key business insights through charts.

## Project Overview
Before building dashboards or machine learning models, it is important to understand the data. This project performs Exploratory Data Analysis (EDA) to examine customer purchasing behavior, payment preferences, referral effectiveness, revenue distribution, and outliers.

## Objectives
- Understand the dataset structure
- Generate descriptive statistics
- Analyze customer purchasing trends
- Detect outliers using the IQR method
- Visualize important business insights
- Summarize key findings

## Dataset
The dataset contains **1,200 retail sales transactions** with **14 columns**, including:
- Order ID
- Date
- Customer ID
- Product
- Quantity
- Unit Price
- Shipping Address
- Payment Method
- Order Status
- Tracking Number
- Items in Cart
- Coupon Code
- Referral Source
- Total Price

## Analysis Performed
### 1. Data Exploration
- Loaded the cleaned dataset
- Examined dataset structure
- Checked data types
- Viewed column names
- Calculated dataset dimensions

### 2. Descriptive Statistics
Calculated:
- Count
- Mean
- Median
- Minimum
- Maximum
- Standard Deviation

### 3. Product Analysis
Analyzed product popularity using frequency counts and visualized the results with a bar chart.

### 4. Payment Method Analysis
Examined customer payment preferences using a pie chart.

### 5. Referral Source Analysis
Compared different customer acquisition channels through a bar chart.

### 6. Revenue Analysis
Calculated the average revenue generated by each product category and visualized the results.

### 7. Outlier Detection
Detected unusually high transaction values using the **Interquartile Range (IQR)** method.

### 8. Correlation Analysis
Computed the correlation between:
- Quantity
- Unit Price
- Total Price

## Key Findings

- The dataset contains **1,200 sales records**.
- Printers were the most frequently ordered product.
- Online payments were the most commonly used payment method.
- Instagram generated the highest number of customer referrals.
- Revenue varied across different products.
- Eight high-value transactions were identified as outliers.
- Quantity and Unit Price both showed positive relationships with Total Price.
